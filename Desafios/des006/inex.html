Tags em HTML

Em HTML5 podemos mover os elementos da lista a seguirç. Passe o mouse pelos termos para descobrir qual tag deverá realizar em cada caso:

Parágrafo
Use a tag <p>
Títulos
Use as tags <h1 até h6>
Imagens
Use a tag <img>
Abreviações
Use a tag <abbr>
Códigos Fonte
Use as tags <pre> e <code>
Listas Númeradas
Use as tags <ol> e <ol>
Listas com Demarcações>
Use as tags <dl>,<dt> e <dd>
