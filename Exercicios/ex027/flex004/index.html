Propriedade:justify-content

A propriedade justify-content serve para definir como os elementos serão distribuídos pelo container pai no eixo principal (main-axis)

*Container com flex-direction: row (padrão)

justify-content: flex-start

posiciona os elementos no eixo main-axis, a partir do main-start

justify-content: flex-end

posiciona os elementos no eixo main-axis, a partir do main-end

justify-content: center

centraliza os elementos no main-axis

justify-content: space-around

divide os main-axis pelo número de elementos e centraliza os elementos nesse espaço

justify-content: space-between

distribui os espaços entre os elementos de forma igual entre os elementos

justify-content: space-evenly

distribui os espaços de forma igual pelo container

Propriedade align-items

Já a propriedade align-items é usada para distribuir os elementos no eixo transversal (cross-axis)

Note que ao definir um tamanho height os elementos por padrão se ajustam ao tamanho do container pai, pois o comportamento padrão dos elementos é align-items: stretch

align-items: flex-start

alinha os elementos a partir do cross-start

align-items: flex-end

alinha os elementos a partir do cross-end

align-items: center

alinha os elementos no centro do cross-axis

Centralizando elementos no centro do container

Como vimos antes podemos combinar ambas as propriedades para centralizar os elementos no centro do container

justify-content:center;
align-items:center;
