Propriedade: order

A propriedade order é utilizada nos elementos filhos de um container flex para definir sua posição na ordem de exibição. Por padrão, todos os elementos possuem order igual a 0, o que significa que serão renderizados na mesma sequência em que aparecem no código HTML. Ao atribuir valores diferentes à propriedade, é possível reorganizar os itens sem alterar a estrutura do documento.

F
como todos os elementos possuem o mesmo nível de order eles aparecem na ordem

Aqui alteramos o elemento 'A' com order: 7;
O elemento 'C' com order: -3;
O elemento 'D' com order: 2;
E o elemento 'F' com order: 1;

Assim os elementos são renderizados seguindo uma ordem crescente, os que tem os menores valores aparecem primeiro. (nosso caso: C:-3, B:0, E:0, F:1, D:2 e A:7)
