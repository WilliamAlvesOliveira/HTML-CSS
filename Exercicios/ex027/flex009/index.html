Flex shrink/grow

No modelo de layout flexível (Flexbox), as propriedades flex-grow e flex-shrink controlam como os itens se expandem ou encolhem dentro do contêiner. Por padrão, flex-grow possui valor 0, o que significa que os itens não irão crescer além do tamanho definido, mesmo que haja espaço disponível. Já o flex-shrink possui valor 1, permitindo que os itens encolham proporcionalmente quando o espaço no contêiner for insuficiente. Assim, ao utilizar o modelo padrão, os elementos mantêm suas dimensões iniciais, encolhendo apenas quando necessário, mas nunca crescendo automaticamente para ocupar espaço extra.

flex-shrink

A propriedade flex-shrink define a capacidade de um item flexível encolher quando o espaço disponível no contêiner é insuficiente. O valor padrão é 1, o que significa que os itens irão reduzir proporcionalmente seu tamanho para se ajustar ao espaço. Se configurarmos flex-shrink: 0;, o item não encolherá, mantendo o tamanho definido pelo flex-basis ou pelas dimensões iniciais, mesmo que isso cause quebra no layout.

flex-grow

Já a propriedade flex-grow controla a capacidade de um item flexível crescer para ocupar o espaço extra disponível no contêiner. O valor padrão é 0, indicando que os itens não irão aumentar além do tamanho definido. Ao atribuir valores maiores, como flex-grow: 1;, o item passa a expandir proporcionalmente em relação aos demais, preenchendo o espaço livre e tornando o layout mais adaptável.

Usando valores variados

Podemos variar os valores das propriedades para definir não apenas se os itens podem ou não diminuir ou crescer, mas também de que forma eles irão se comportar nesse processo.

Ao utilizarmos valores variados na propriedade flex-shrink, definimos diferentes proporções de encolhimento entre os itens dentro de um contêiner flex. Isso significa que, quando o espaço disponível é reduzido, os elementos com valores maiores irão encolher mais rapidamente em relação aos que possuem valores menores. Por exemplo, um item com flex-shrink: 2; diminuirá o dobro em comparação a outro com flex-shrink: 1;, permitindo controlar de forma precisa como cada elemento se adapta em cenários de falta de espaço.

items com flex-basis:150px;
shrink:0
shrink:1
shrink:2
shrink:3
Notem que até o limite do conteúdo o item que possui flex-shrink: 2; diminuirá o dobro do que tem o flex-shrink: 1;

O mesmo princípio se aplica à propriedade flex-grow, que define a proporção de crescimento dos itens dentro de um contêiner flex quando há espaço extra disponível. Ao atribuir valores variados, controlamos quanto cada elemento irá expandir em relação aos demais. Por exemplo, um item com flex-grow: 2; crescerá o dobro em comparação a outro com flex-grow: 1;, ocupando uma parte maior do espaço livre. Dessa forma, conseguimos distribuir dinamicamente o espaço excedente e tornar o layout mais equilibrado e adaptável.

item com flex-basis: 50px;
grow:0
grow:1
grow:2
grow:3
Neste exemplo, os itens com valores mais altos de flex-grow expandem proporcionalmente mais do que o item configurado com flex-grow: 1;, ocupando uma parte maior do espaço disponível no contêiner.

ShortHand flex

Temos a propriedade flex que é uma shorthand para as propriedades flex-grow,flex-shrink e flex-basis

flex-grow: 2;
flex-shrink: 1;
flex-basis: 100px;
essas propriedades podem ser simplificadas a partir da propriedade flex.
flex: 2 1 100px;

Valores da propriedade flwx

A propriedade flex possuí alguns valores que podem ser aplicados, esses valores são os seguintes

flex: initial; = (flex:0 1 auto)

flex: none; = (flex: 0 0 auto)

flex: auto; = (flex: 1 1 auto)

caso usamos flex seguido apenas por um valor, isso vai configurar apenas o flex-grow deste elemento, e manter as outras com o comportamento padrão
