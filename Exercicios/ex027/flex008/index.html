Propriedade flex-basis

A propriedade flex-basis define o tamanho inicial de um elemento flexível antes da distribuição de espaço extra dentro do container. Ela determina a largura ou altura base do item (dependendo da direção do eixo principal), servindo como referência para o cálculo do layout. Por padrão, o valor de flex-basis é auto, o que significa que o tamanho inicial do elemento será definido pelo seu conteúdo ou por outras propriedades de dimensão, como width e height.

Neste exemplo, os elementos foram definidos com flex-basis: 100px;, o que estabelece um tamanho inicial de 100px para cada item. Esse valor será respeitado enquanto houver espaço disponível no container, podendo ser ajustado caso o layout exija redistribuição.

Quando não há espaço suficiente no container para acomodar todos os elementos com o tamanho definido em flex-basis, o Flexbox redistribui o espaço de acordo com as propriedades flex-grow e flex-shrink. Nesse cenário, os itens podem encolher proporcionalmente (se flex-shrink estiver habilitado) ou crescer para ocupar o espaço disponível (se flex-grow for maior que 0). Assim, o valor de flex-basis funciona apenas como ponto de partida, podendo ser ajustado dinamicamente para manter o layout flexível.

Neste exemplo, os elementos também possuem flex-basis: 100px;, porém, como não há espaço suficiente, eles se ajustam proporcionalmente ao tamanho disponível no container pai.

Caso o container pai tenha a propriedade flex-wrap habilitada, o flex-basis continua sendo aplicado normalmente. Nesse cenário, os elementos respeitam o tamanho inicial definido e, se não houver espaço suficiente em uma única linha, eles serão distribuídos em múltiplas linhas ou colunas conforme o comportamento do flex-wrap.
