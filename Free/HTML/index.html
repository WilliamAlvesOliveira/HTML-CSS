HTML & CSS que parecem JavaScript

Nesta página vamos explorar efeitos e comportamentos interativos criados usando apenas HTML e CSS — recursos que, por muitos anos, só eram possíveis com JavaScript.

O objetivo aqui não é substituir o JavaScript, mas mostrar como os navegadores modernos já resolvem sozinhos muitos problemas de interface, como sliders, animações, transições e navegação.

⚠️ Observação: alguns recursos utilizados dependem de navegadores modernos. Em browsers antigos, a página continua funcional, mas sem alguns efeitos visuais (princípio de progressive enhancement).

O que vamos explorar nesta página

  • Layouts responsivos
  • Cards com proporção fixa
  • Slider / carrossel
  • Fade-in on scroll
  • Abas (tabs)
  • Modais
