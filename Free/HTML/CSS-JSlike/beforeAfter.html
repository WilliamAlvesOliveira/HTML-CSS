Pseudo-elemento

::BEFORE / ::AFTER

Os pseudo-elementos ::before e ::after em CSS são usados para inserir conteúdo gerado dinamicamente antes ou depois do conteúdo real de um elemento, sem a necessidade de alterar o HTML. Eles são muito úteis para adicionar ícones, símbolos, ornamentos visuais ou até mesmo textos auxiliares, mantendo o código mais limpo e sem poluir a marcação. Por exemplo, podem ser aplicados para criar efeitos decorativos em botões, adicionar aspas estilizadas em citações ou inserir elementos gráficos em listas. A aplicabilidade é ampla, já que permitem enriquecer a interface e melhorar a experiência do usuário com recursos visuais e funcionais, tudo apenas via CSS.
